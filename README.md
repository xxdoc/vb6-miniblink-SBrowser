# SBrowser OCX 控件
## 使用VB6开发，暂测试未遇到问题。
## 使用最新 miniblink 190717 版本，已完成除了定制版外文档上写着的API接口（部分未测试，如使用接口遇到问题可发issues）。
## 如果想要更换其它版本，只要把目录下的node.dll更换即可（node.dll必须和SBrowser_G.ocx在同一目录下）
